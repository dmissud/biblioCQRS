📚 BiblioCQRS - API de Commande

Le Projet BiblioCQRS

BiblioCQRS est un système de gestion de bibliothèque moderne implémenté en utilisant l'architecture CQRS (Command Query Responsibility Segregation) et Event Sourcing.

L'architecture sépare strictement les opérations en deux parties distinctes : l'écriture (Command) et la lecture (Query), qui communiquent de manière asynchrone pour garantir performance et scalabilité.

Ce composant : Command

Ce module (biblio-command-boot) est responsable de la partie écriture.

Il reçoit les intentions d'action (commandes) telles que l'ajout d'un livre, son emprunt ou son retour. Il valide ces commandes selon les règles métier du domaine et, si elles sont valides, génère des événements persistés dans l'Event Store.
